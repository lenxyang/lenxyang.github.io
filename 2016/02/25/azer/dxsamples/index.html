DirectX Samples

Feb 25, 2016 | Azer3D | Hits

Direct3D SDK 当中有很多 demo 都是优秀而是用的，在学习图形学过程中非常值得实现

Instancing10

使用 Multi-stream 的方法传递 instance draw 的参数，单单这一点就非常有益。此外，他的植被渲染也很有启发。

ShadowVolumn10

Graphics 的经典算法，应该实现以下。既然 DirectX 有完整实例，使用自己的引擎实现。

CubeMapGS (Direct3D 10)

HDAO10.1

SSAO 算法几乎成为标配, 看看 AMD 的完整实例吧

AdvancedParticles

这个实例讲解了一个完全由 Render Pipeline 完成的例子系统与其他物体交互的实例

MontionBlur10

基于 Geometry Shader 实现的 Moniton Blur，在开始阶段略有些复杂。可以参考 Direct3D 9 的 PixelMontionBlur。

Shadowmap

Shadowmap 有一系列的实例
