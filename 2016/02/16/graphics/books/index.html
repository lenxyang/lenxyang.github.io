Direct3D

Introduction to 3D Game Programming with DirectX 11

介绍 Direct3D 的最经典书籍，很多办了，最新已经到了 DirectX 12。每一个例子都应该实现一下。

Advanced Animation with DirectX

动画效果的入门书，内容不错，可以读一读，深度比 Introduction to 3D Game 略深（动画部分）

Character Animation with Direct3D

介绍角色动画的好书，每一个例子都值得实现一遍

Practical Rendering and Computation with Direct3D 11

本书的着眼点是 Rendering Pipeline，更多的介绍了 Tessellation, Computation, Deferred Shader, Multithread Rendering 这些东西。

Real-Time 3D Rendering with DirectX and HLSL

还没有读

Advanced 3D Game Programming with DirectX 10

本书的着眼点是 Game Programming，介绍了几乎游戏引擎需要的各个技术点，跟 DirectX 渲染的书籍相比，他还介绍了想 DXSournd, DXinput, Scene Management 这些东西。

Direct3D Rendering Cookbook

比较简单的 Direct3D 图书，使用 C# 实现， 基于 SharpDX。 这本书有一些有深度的 demo，包括如何使用 BulletDX 引擎绘制 occean，对于了解一些东西还是很有好处的，此外也包含了 Defer Shader 和 Multithread Render。
