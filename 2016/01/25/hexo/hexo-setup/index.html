Trouble Shooting

运行 hexo 是提示 “/usr/bin/env: node: No such file or directory”

最简单的办法是

hexo deploy: ERROR Deployer not found: github

hexo 从 3.0 以后，type 由 github 变为了 git。毕竟主要有一个 git repository 就可以部署了，并非 github 不可。
