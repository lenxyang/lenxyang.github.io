Recuse Port

默认情况下，服务端口关闭（如果有通信的话）会在一段时间内处于 TIME_WAIT 状态，这种状态会阻止服务器重新 (bind 失败), 因此对于服务器而言，减少 TIME_WAIT 的时间非常必要。对于 RHEL 来说可能通过更改配置文件 /etc/sysctl.conf 来达到目的

控制客户端端口分配范围

客户端端口的分配范围是需要配置的，否则很有可能自动分配的端口占用了服务器预期使用的端口导致服务器无法启动（在服务器重启阶段端口有可能被占用）。
