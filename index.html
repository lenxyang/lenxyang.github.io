Decal

Decal

Tessellation based method

Tessellation based 的方法能够不影响固有程序流程做到 decal, 调优方面的工作基本全部都在 hlsl 内部，这种方法有非常好的适用性。

此方法的流程如下

  • 确定 hit position: 找到与 mesh 匹配的点
  • 在 ds 当中将 decal texture 映射到 hit position 上
  • 而后计算 displacement

Practical Rendering and Computation with Direct3D 11

gcc

GCC Tricks

去掉 debug info

一般情况下 debug info 是不需要的，发布时可以包括两个版本带有调试信息和不带调试信息的，上线使用不带调试信息的，如果出现 core 在使用带有调试信息的追问题。（某些时候带宽真的是问题，发布的程序越小越好）

连接静态 libgcc, libstdc

libgcc 是非常基础的库，他基本与编译器是绑定的，如果部署的机器 gcc 版本较低，很有可能会因为 libgcc 的版本没办法运行
